📋 Notes & Reminders
  • Cover images: Upload the FRONT COVER ONLY as a portrait (tall) image — PNG or JPG. No full wraparound covers. Upload to the covers/ folder in GitHub and set the filename here (e.g. covers/my-book.png).
  • Status: Use the dropdown to set where each book sits in the pipeline — Published, Coming Soon, Fighting Back, Genesis, A Dark Thought, or Free.
  • Free titles: Paste the complete text of the story or essay into the Full Text box. Paragraph breaks are preserved automatically.
  • Tags: Separate tags with commas. No # symbol needed here (e.g. dark, british, comedy, short story).
  • Collective titles: If the book is a Pitch Dark Collective collaboration, tick the Collective box.
  • After making changes: Click Download Updated Catalog at the bottom, then upload the file to GitHub as normal.
When you're done making changes
Download the file, then upload it to GitHub (Add file → Upload files → Commit changes).
The live site updates within a minute or two.
